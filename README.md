# Zappp Website - Based on Angular 2, Bootstrap 4 and Webpack
Using Ng2 Admin template made by [Akveo team](http://akveo.com/).

## Features
* TypeScript
* Webpack
* Responsive layout
* High resolution
* Bootstrap 4 CSS Framework
* Sass
* Angular 2
* jQuery
* Charts (Chartist, Chart.js)
* Maps (Google, Leaflet, amMap)
* and many more!
